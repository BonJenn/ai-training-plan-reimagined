# AI Training Plan — Reimagined

A modern, responsive landing-page concept for an adaptive AI endurance coaching product. The redesign keeps the original product promise while simplifying the story, elevating the visual system, and making the core idea interactive.

**Live demo:** [ai-training-plan-reimagined.vercel.app](https://ai-training-plan-reimagined.vercel.app)

## Highlights

- Immersive athlete-first hero
- Interactive readiness and workout adaptation demo
- Responsive layouts for mobile, tablet, and desktop
- Accessible semantic structure and keyboard states
- Product storytelling around unified data and explainable AI coaching
- Lightweight Next.js implementation with no backend dependencies

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React
- Vercel

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes

This is an independent redesign exercise built from scratch. The waitlist form is a front-end demo and does not transmit or store email addresses.
